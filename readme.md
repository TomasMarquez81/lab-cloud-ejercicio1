Lo primero que hacemos es crear la built del proyecto, que nos crea la carpeta dist con todo lo necesario para que funcione el proyecto.

Después de crear la built, creamos un repositorio en github y subimos el proyecto.

`bash
// Para subir el proyecto a github hacemos lo siguiente:
// 1. git init
// 2. git add .
// 3. git commit -m "first commit"
// 4. git branch -M main
// 5. git remote add origin <url_del_repositorio>
// 6. git push -u origin main
`
